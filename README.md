# Wordpress Bedrock CiviCRM Upsun Template

## Local Development

Clone this repo

composer install

cp .env.example .env

update env values appropriately

Install WP and CiviCRM as normal.


## Upsun

Push to [Upsun](https://upsun.com)

Add Upsun Environment variables for CIVICRM_SIGN_CREDS, CIVICRM_CRED_KEYS, CIVICRM_SITE_KEY, CIVICRM_DEPLOY_ID

Install WP and CiviCRM as normal.
